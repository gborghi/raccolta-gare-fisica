# OII 2016 1° Livello — Quesito 7

Un sistema costituito da un gas perfetto compie il ciclo termodinamico rappresentato in figura. La trasformazione $1\to2$ è un'isoterma. La trasformazione $2\to3$ è un'isocora e la trasformazione $3\to1$ è un'adiabatica. Durante l'intero ciclo viene compiuto un lavoro $L=10\,\text{J}$. Durante la trasformazione $2\to3$, l'energia interna diminuisce di $20\,\text{J}$. Durante la trasformazione $3\to1$, viene fatto un lavoro di $20\,\text{J}$ sul sistema.

| | $Q$ | $L$ | $\Delta U$ |
|---|---|---|---|
| $1\to2$ | | | |
| $2\to3$ | | | $-20\,\text{J}$ |
| $3\to1$ | | $-20\,\text{J}$ | |
| Ciclo | | $+10\,\text{J}$ | |

Quanto calore viene somministrato al sistema nella trasformazione $1\to2$?

(A) $0$
(B) $10\,\text{J}$
(C) $20\,\text{J}$
(D) $30\,\text{J}$
(E) $40\,\text{J}$
