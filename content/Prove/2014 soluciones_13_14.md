## Problema 1

Problema 1 – El vuelo del Apolo XI

a) Si el satélite describe una órbita circular, la fuerza gravitatoria es la fuerza centrípeta que
origina dicho movimiento

Fuerza gravitatoria: $F_g = G\dfrac{M\cdot m}{r^2}$

Fuerza centrípeta: $F_c = m\dfrac{v^2}{r}$

$$G\frac{M\cdot m}{r^2} = m\frac{v^2}{r} \quad\Rightarrow\quad v = \sqrt{G\frac{M}{r}}$$

$G = 6{,}67\cdot10^{-11}\ \dfrac{\text{N m}^2}{\text{kg}^2}$

$M_T = 5{,}98\cdot10^{-24}\ \text{kg}$

$r = R_T + h = 6{,}38\cdot10^6\ \text{m} + 1{,}85\cdot10^5\ \text{m} = 6{,}57\cdot10^6\ \text{m}$

$$v = \sqrt{6{,}67\cdot10^{-11}\ \frac{\text{N m}^2}{\text{kg}^2}\cdot\frac{5{,}98\cdot10^{-24}\ \text{kg}}{6{,}57\cdot10^6\ \text{m}}} = 7{,}79\cdot10^3\ \frac{\text{m}}{\text{s}}$$

$$\boxed{v = 7{,}79\cdot10^3\ \frac{\text{m}}{\text{s}}}$$

b)

La suma de los campos en el punto P es cero:

$$g_T + g_L = 0 \quad\Rightarrow\quad g_T\,\vec{\imath} - g_L\,\vec{\imath} = 0 \quad\Rightarrow\quad g_T = g_L$$

Los módulos de los campos son iguales:

$g_T = G\dfrac{M_T}{d_T^{\,2}}$ ; $g_L = G\dfrac{M_L}{d_L^{\,2}}$ ; $d_T = r_L - d_L$ ; $M_L = \dfrac{M_T}{81}$

$$G\frac{M_T}{d_T^{\,2}} = G\frac{M_L}{d_L^{\,2}} \quad\Rightarrow\quad G\frac{M_T}{d_T^{\,2}} = G\frac{M_T}{81\cdot d_L^{\,2}}$$

$$\frac{1}{d_T^{\,2}} = \frac{1}{81\cdot d_L^{\,2}} \quad\Rightarrow\quad d_T^{\,2} = 81\cdot d_L^{\,2}$$

$$d_T = 9\cdot d_L \quad\Rightarrow\quad d_T = 9\,(r_L - d_T) \quad\Rightarrow\quad d_T = \frac{9\cdot r_L}{10}$$

$r_L = 3{,}80\cdot10^8\ \text{m} \quad\Rightarrow\quad d_T = 3{,}42\cdot10^8\ \text{m}$

$$h = d_T - R_T \qquad h = 3{,}42\cdot10^8\ \text{m} - 6{,}38\cdot10^6\ \text{m} = 3{,}36\cdot10^8\ \text{m}$$

$$\boxed{h = 3{,}36\cdot10^8\ \text{m}}$$

El potencial es la suma de los potenciales del campo de la Tierra y del campo de la Luna.

El potencial gravitatorio es siempre negativo y su valor es: $V = -G\dfrac{M}{r}$

Potencial del campo terrestre: $V_T = -G\dfrac{M_T}{d_T}$

Potencial del campo lunar: $V_L = -G\dfrac{M_L}{d_L}$

$$V = -G\frac{M_T}{d_T} - G\frac{M_L}{d_L}$$

$M_L = \dfrac{M_T}{81}$ ; $d_T = 3{,}42\cdot10^8\ \text{m}$ ; $d_L = r_L - d_T = 3{,}80\cdot10^8\ \text{m} - 3{,}42\cdot10^8\ \text{m} = 3{,}80\cdot10^7\ \text{m}$

$$V = -G\cdot M_T\left(\frac{1}{d_T} + \frac{1}{81\cdot d_L}\right)$$

$$V = -6{,}67\cdot10^{-11}\ \frac{\text{N}\cdot\text{m}^2}{\text{kg}^2}\cdot5{,}98\cdot10^{24}\ \text{kg}\left(\frac{1}{3{,}42\cdot10^8\ \text{m}} + \frac{1}{81\cdot3{,}80\cdot10^7\ \text{m}}\right)$$

$$\boxed{V = -1{,}30\cdot10^6\ \frac{\text{J}}{\text{kg}}}$$

c)

El movimiento entre P1 y P2 no es uniformemente acelerado puesto que g no es constante.
Como solamente actúan fuerzas gravitatorias la energía mecánica se conserva:

$$E_{c1} + U_1 = E_{c2} + U_2$$

PUNTO P1.- Actúan el campo gravitatorio terrestre y el campo gravitatorio lunar y el cohete
tiene una velocidad $v_1 = 122\ \dfrac{\text{m}}{\text{s}}$

El potencial de un punto es la energía potencial de la unidad de masa en ese punto:

Energía potencial: $V_1 = \dfrac{U_1}{m} \Rightarrow U_1 = V_1\cdot m$, $V_1 = -1{,}30\cdot10^6\ \dfrac{\text{J}}{\text{kg}}$ (calculado en el apartado
anterior)

Energía cinética: $E_{c1} = \dfrac{1}{2}m\cdot v_1^2 \quad\Rightarrow\quad v_1 = 122\ \dfrac{\text{m}}{\text{s}}$

PUNTO P2.- El campo gravitatorio terrestre es despreciable

Energía potencial: $U_2 = -G\dfrac{M_L\cdot m}{r_2} \quad\Rightarrow\quad r_2 = R_L + h = 1{,}74\cdot10^6\ \text{m} + 8{,}00\cdot10^4\ \text{m}$

Energía cinética: $E_{c2} = \dfrac{1}{2}m\cdot v_2^2 \quad\Rightarrow\quad v_2 = ?$ (es lo que pide el problema)

$$V_1\cdot m + \frac{1}{2}m\cdot v_1^2 = -G\frac{M_L\cdot m}{r_2} + \frac{1}{2}m\cdot v_2^2$$

$$-1{,}30\cdot10^6\ \frac{\text{J}}{\text{kg}} + \frac{1}{2}\left(122\ \frac{\text{m}}{\text{s}}\right)^2 = -6{,}67\cdot10^{-11}\ \frac{\text{N}\cdot\text{m}^2}{\text{kg}^2}\cdot\frac{5{,}98\cdot10^{24}\ \text{kg}}{81\cdot1{,}82\cdot10^6\ \text{m}} + \frac{1}{2}v_2^2$$

$$\boxed{v_2 = 1{,}68\cdot10^3\ \frac{\text{m}}{\text{s}}}$$

## Problema 2

Problema 2

a) $v_{x0} = \sqrt{\dfrac{2eV}{m}} = 2{.}3\times10^7\ \text{m/s}$

b) $t_1 = \dfrac{l_1}{v_{x0}} = 2{.}17\times10^{-9}\ \text{s}$

c) $t_2 = \dfrac{l_2}{v_{x0}} = 4{.}35\times10^{-9}\ \text{s}$

d) $\Delta y_p = \dfrac{1}{2}a t_1^2$ ; $a = \dfrac{eE}{m} = \dfrac{eV_p}{md}$

$\Delta y_f = \Delta y - \Delta y_p = v_{yf}\,t_2$ ; $v_{yf} = a t_1 = \dfrac{eV_p}{md}t_1$

$\Delta y = \Delta y_p + \Delta y_f = \dfrac{1}{2}\dfrac{eV_p}{md}t_1^2 + \dfrac{eV_p}{md}t_1 t_2$

$V_p = \dfrac{\Delta y}{\dfrac{et_1}{md}\left(\dfrac{1}{2}t_1 + t_2\right)} = 724\ \text{V}$

e) $E = \dfrac{V_p}{d} = 24{.}13\ \text{kV/m} = 24{.}13\ \text{kN/C}$

f) $\Delta y_p = \dfrac{1}{2}\dfrac{eV_p}{md}t_1^2 = 10^{-2}\ \text{m} = 1\ \text{cm}$

g) $\alpha = \arctan\dfrac{v_{yf}}{v_{x0}} = \arctan 0{.}4 = 0{.}38\ \text{rad} = 21{.}8^\circ$

$v_{yf} = \dfrac{eV_p}{md}t_1 = 9{.}2\times10^6\ \text{m/s}$

## Problema 3

Problema 3

a) Conservación de la energía para el sistema muelle-flecha:

$$\frac{1}{2}K x^2 = \frac{1}{2}m v_0^2 \quad\Rightarrow\quad v_0 = \sqrt{\frac{500}{0{,}05}}\cdot0{,}1 = 10\ \text{m/s}$$

Movimiento de la flecha hasta la manzana:

- eje x : movimiento uniforme con velocidad $v_{0x}$
- eje y : movimiento uniformemente acelerado con velocidad inicial $v_{0y}$ y aceleración la
gravedad $a_y = -9{,}8\ \text{m/s}^2$

$$v_{fy} = v_{0y} + a_y t \quad\Rightarrow\quad 0 = v_{0y} - 9{,}8\,t \quad\Rightarrow\quad t = \frac{v_{0y}}{9{,}8}$$

$$y = y_0 + v_{0y} t + \frac{1}{2}a_y t^2 \quad\Rightarrow\quad 2 = v_{0y}\frac{v_{0y}}{9{,}8} - \frac{1}{2}\,9{,}8\left(\frac{v_{0y}}{9{,}8}\right)^2$$

Luego $v_{0y} = 6{,}26\ \text{m/s}$

$$v_{0y} = v_0\,\text{sen}\,\theta \quad\Rightarrow\quad \text{sen}\,\theta = \frac{6{,}26}{10} \quad\Rightarrow\quad \theta = 38{,}8^\circ$$

$$d_x = v_{0x}\cdot t = v_{0x}\cdot\frac{v_{0y}}{9{,}8} = 4{,}98\ \text{m}$$

b) El movimiento del conjunto flecha+manzana acaba en la posición $(x'_f, 0)$ después de un
tiempo $t'$. Vuelve a ser uniformemente acelerado en la dirección y con $y_0 = 2$ m, $v'_{0y} = 0$, $a_y = -9{,}8$
m/s$^2$ ; en la dirección x se trata de un movimiento uniforme con velocidad $v'_x$.

$$y' = y'_0 + v'_{0y} t + \frac{1}{2}a_y t'^2 \quad\Rightarrow\quad 0 = 2 - \frac{1}{2}\,9{,}8\,t'^2 \quad\Rightarrow\quad t' = 0{,}64\ \text{s}$$

$$x'_f = x'_i + v'_x t' \quad\Rightarrow\quad 6{,}25 = 4{,}98 + v'_x t' \quad\Rightarrow\quad v'_x = \frac{1{,}27}{0{,}64} = 1{,}98\ \text{m/s}$$

Conservación del momento lineal en la dirección x:

$$m\cdot v_{0x} = (m + M)\cdot v'_x$$

Donde m es la masa de la flecha y M la de la manzana.

$$0{,}05\cdot10\cos(38{,}8) = (0{,}05 + M)\cdot1{,}98 \quad\Rightarrow\quad M = 0{,}15\ \text{kg}$$
