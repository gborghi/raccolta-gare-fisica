# INJSO 2017 — Problema 22

|---|---|
| 1. Cinnabar | I. Oxidation and reduction |
| 2. Zincblende | II. Oxidation |
| 3. Hematite | III. Electrolysis |
| 4. Galena | IV. Reduction |

- (a) 1-I, 2-II, 3-IV, 4-II
- (b) 1-II, 2-I, 3-IV, 4-I
- (c) 1-2, 3-II, 4-III, 4-IV
