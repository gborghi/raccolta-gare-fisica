# INJSO 2019 — Problema 5

Study the following three reactions:

(i) $\text{CO}_2 + \text{H}_2\text{S} \rightarrow (\text{CH}_2\text{O})_n + 2\text{S} + \text{H}_2\text{O}$

(ii) $\text{CO}_2 + \text{S} + \text{H}_2\text{O} \rightarrow (\text{CH}_2\text{O})_n + \text{H}_2\text{SO}_4$

(iii) $\text{CO}_2 + \text{H}_2\text{O} \rightarrow \text{C}_6\text{H}_{12}\text{O}_6 + \text{O}_2$

Which reaction/s represent/s autotrophic nutrition?

- (A) (iii) only
- (B) (i) and (iii) only
- (C) (ii) and (iii) only
- (D) (i), (ii) and (iii)
