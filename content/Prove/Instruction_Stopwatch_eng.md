## Problema 1








Stopwatch model FS-810
ENGLISH LANGUAGE
Button A — start/stop.
Button B — Split/Lap memory (stage memory).
Button C — Record splits / Reset.
Button D — Mode (do not use).

Recording split (stage) times
1. At the start, press Button A (time = 0).
2. At the end of Stage 1, press Button C (the stopwatch memory stores the
time for Stage 1). At the end of Stage 2, press Button C again (the
memory stores the cumulative time for Stages 1–2), and so on.
3. After completing 10 stages, press Button A (stop the stopwatch).

Reading split (stage) times
1. Press Button B (switch to split-time reading mode).
2. Press Button C (the display will show the cumulative time for the
stages).
3. By pressing Button C repeatedly, you can read the cumulative times from Stage 1 to Stage 10.
(In the figure: Stage 8 and the cumulative time for 8 stages is 7.13 s.)
4. To return to split-time recording mode, press Button B.
5. To clear the split memory, press Button C.

If necessary, you can repeat the split recording and reading process.
