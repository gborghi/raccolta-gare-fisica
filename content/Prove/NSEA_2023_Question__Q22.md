# NSEA 2023 — Problema 22

For what values of $a$ and $b$ is the following function continuous at $x = 2$?

$$f(x) = \begin{cases} ax^2 + bx + 1 & \text{if } x < 2 \\ 3 & \text{if } x = 2 \\ bx^2 + ax + 11 & \text{if } x > 2 \end{cases}$$

- (a) $a = 1,\, b = 9$
- (b) $a = 2,\, b = 12$
- (c) $a = 1,\, b = 3$
- (d) $a = 3,\, b = 3$
