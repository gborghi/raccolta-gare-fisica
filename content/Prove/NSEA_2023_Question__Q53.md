# NSEA 2023 — Problema 53

The true statement(s) for the function

$$f(x) = \begin{cases} \dfrac{\sin x}{x} & \text{if } x \neq 0 \\ 0 & \text{if } x = 0 \end{cases}$$

is/are

- (a) $\lim_{x \to 0} f(x)$ exists
- (b) $\lim_{x \to 0} f(x)$ does not exist
- (c) $f$ is differentiable at $x = 0$
- (d) $f$ has removable discontinuity at $x = 0$
