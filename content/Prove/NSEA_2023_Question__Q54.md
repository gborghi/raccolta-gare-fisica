# NSEA 2023 — Problema 54

Consider the differential equation $x\dfrac{dy}{dx} - 3y = x^3$. Which of the following is/are true?

- (a) $3x^3$ is an integrating factor of this equation
- (b) $y = \dfrac{x^3}{6} + \dfrac{2023}{x^3}$ is a solution of this equation
- (c) $y = \dfrac{x^3}{6} - \dfrac{8}{2023x^3}$ is a solution of this equation
- (d) The equation is a homogeneous differential equation
