# NSEA 2024 — Problema 25

Consider the following system of linear equations:

$$kx + y + z = 2023$$
$$k^2 x + ky + z = 2024$$
$$k^3 x + k^2 y + z = 2025$$

Which of the following is true?

- (a) The system has a unique solution for $k = 1$
- (b) The system has a unique solution for any value of $k$
- (c) The system is always inconsistent for any value of $k$
- (d) The system has infinitely many solutions for $k = 1$
