# NSEJS 2023 — Problema 49

In a classroom, students were taught typical mammalian characters along with the names of Orders and representative examples. In the Table given below, column 1 includes the names of examples or Orders whereas column 2 shows related characteristics.

| Order / Representative example | Characteristics |
| --- | --- |
| 1. Lagomorpha | (i) First finger clawed, tail enclosed in an interfemoral membrane. |
| 2. Microchiroptera | (ii) Toothless and Polyembryony. |
| 3. *Armadillo* | (iii) Baleen. |
| 4. Proboscidea | (iv) Incisors open-rooted and continue to grow throughout life. |

Choose the option(s) that has the correct match in the above table.

- (a) 1 → (iv)
- (b) 2 → (i)
- (c) 3 → (ii)
- (d) 4 → (iii)
