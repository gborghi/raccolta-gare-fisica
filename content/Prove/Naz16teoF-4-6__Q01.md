# OII 2016 Nazionale Teorica — Problema 1

P1 Lo stucco nel secchiello

1 Equilibrio – Dimostrazione:
La forza necessaria per garantire l’equilibrio è minore di quella massima (di distacco).
Forza d’attrito:
$A = m_1 g = 8.32 \leq 8.34 \leq 8.35\ \text{N}$

2 Valore minimo dell’intervallo $\Delta t$:
$(\Delta t)_\text{min} = \dfrac{m_2 v}{(\alpha\mu_\text{s} M - \alpha m_1 - m_2)\, g} = 0.79 \leq 0.82 \leq 0.86\ \text{s}$

3 Velocità del secchiello $\vec{V}$:
$V = \dfrac{m_2 v}{m_1 + m_2 + M} = 0.551 \leq 0.554 \leq 0.557\ \text{m s}^{-1}$
verso il basso.

4 Accelerazione del secchiello $\vec{a}$:
$a = \dfrac{(m_1 + m_2 - \mu_\text{d} M)\, g}{m_1 + m_2 + M} = -1.02 \leq -0.99 \leq -0.96\ \text{m s}^{-2}$
verso l’alto.

5 Velocità del secchiello $\vec{V}'$:
$V' = \dfrac{m_2 v}{m_1 + m_2 + m_3/2 + M} = 0.507 \leq 0.510 \leq 0.513\ \text{m s}^{-1}$
verso il basso.

6 Accelerazione del secchiello $\vec{a}'$:
$a' = \dfrac{(m_1 + m_2 - \mu_\text{d} M)\, g}{m_1 + m_2 + m_3/2 + M} = -0.93 \leq -0.91 \leq -0.89\ \text{m s}^{-2}$
verso l’alto.
