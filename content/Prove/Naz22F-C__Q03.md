# OII 2022 Nazionale Teorica — Problema 3

P3 Ciclo termodinamico

1 Pressione del gas in B

$$p_B = p_A\,(V_A/V_B)^\gamma = (104.7 \leq 105 \leq 105.3)\ \text{kPa}$$

Espressione · Valore numerico

2 Equazione di trasformazione AB

$$m = (p_B - p_A)/(V_B - V_A) = (-155.8 \leq -0.155 \leq -154.2)\ \text{kPa}\,\text{dm}^{-3}$$

$$q = (p_A V_B - p_B V_A)/(V_B - V_A) = (3.799 \leq 3.825 \leq 3.851)\ \text{kPa}$$

Espressione · Valore numerico

3 Temperature massima e minima

$$T_\text{max}:\quad V_X = -\frac{q}{2m} = (12.20 \leq 12.34 \leq 12.48)\ \text{dm}^3$$

$$p_X = \frac{q}{2} = (1.901 \leq 1.912 \leq 1.925)\ \text{kPa}$$

$$T_\text{min}:\quad V_B = 24\ \text{dm}^3$$

$$p_B = (104.7 \leq 105 \leq 105.3)\ \text{kPa}$$

Espressione · Valore numerico

4 Stato Y in cui termina assorbimento di calore

$$V_Y = -\frac{5q}{8m} = (15.24 \leq 15.42 \leq 15.60)\ \text{dm}^3$$

$$p_Y = \frac{3q}{8} = (1.424 \leq 1.434 \leq 1.444)\ \text{kPa}$$

Espressione · Valore numerico

5 Rendimento del ciclo.

$$\eta = \frac{L_\text{netto}}{Q_\text{ass}} = (50.5 \leq 52.3 \leq 54.1)\ \%$$

Espressione · Valore numerico

6 Rendimento del ciclo di Carnot

$$\eta_C = 1 - \frac{T_\text{min}}{T_\text{max}} = 1 + \frac{4m}{q^2}\,p_B V_B = (89.08 \leq 89.32 \leq 98.56)\ \%$$

Espressione · Valore numerico
