**Material experimental distribuído**

Para esta experiência você recebeu o seguinte material:

- Um perfil em L;
- Parafuso com porca;
- Três arruelas;
- Barbantes com 50 cm, 80 cm e 110 cm de comprimento;
- Uma fita métrica de costureira.

Você também usará um cronômetro e fita crepe, que serão distribuídos pelo professor/instrutor.

**Breve resumo teórico**

Um pêndulo simples é constituído por um corpo de massa $m$ suspenso por um fio de comprimento $l$. Esse corpo pode oscilar em torno da posição de equilíbrio, com período de oscilação $T$ (o período $T$ é definido como o tempo em que o pêndulo realiza uma oscilação completa, ou seja, sai do ponto extremo e volta a esse ponto). Nesta experiência você irá determinar o período do pêndulo, considerando diferentes comprimentos ($l$) e diferentes massas ($m$). Para isso deve-se colocar o pêndulo a oscilar, para pequenos ângulos $\theta$.

## Problema 1







**A — Procedimento experimental I: dependência do período com o comprimento**

Usando uma arruela como a massa $m$, solte o pêndulo de um dos extremos (use um ângulo $\theta$ pequeno). Como não é fácil medir o período de apenas uma oscilação — o período $T$ é definido como o tempo transcorrido desde o instante que o pêndulo sai de sua mão até voltar —, você irá medir para cada pêndulo, com o cronômetro, o **tempo de 10 oscilações completas**. Comece usando o pêndulo com comprimento de 30 cm; faça essa medida três vezes e anote os resultados no Registro de dados experimentais I.

Repita o procedimento usando o pêndulo com comprimento de 60 cm e 90 cm, e anote os resultados no Registro de dados experimentais I.

> **Observação importante:** Você deve fazer com que o ângulo $\theta$ seja o menor possível.

**Registro dos dados experimentais I**

| Comprimento do pêndulo | $t_1$ (s) | $t_2$ (s) | $t_3$ (s) |
|---|---|---|---|
| $l = 30\,\text{cm}$ (uma arruela) | | | |
| $l = 60\,\text{cm}$ (uma arruela) | | | |
| $l = 90\,\text{cm}$ (uma arruela) | | | |

*(Tempo referente a 10 oscilações completas em cada medida.)*

*Obs:* Pode ser difícil realizar experimentos com os valores exatos dos comprimentos pedidos (30 cm, 60 cm e 90 cm). Se encontrar alguma dificuldade, faça a experiência com valores próximos aos pedidos e os registre ao lado dos valores acima.

## Problema 2







**B — Procedimento experimental II: dependência do período com a massa**

(a) Usando o pêndulo de comprimento $l = 90\,\text{cm}$, amarre **duas arruelas** e meça o tempo de 10 oscilações, como feito no Procedimento Experimental I. Faça essa medida três vezes e anote os resultados no Registro de dados experimentais II.

(b) Repita o procedimento (a) usando **três arruelas** e anote os resultados no Registro de dados experimentais II.

(c) Copie na tabela abaixo os dados da tabela anterior (Registro dos dados experimentais I) referentes ao comprimento $l = 90\,\text{cm}$ (uma arruela).

**Registro dos dados experimentais II**

| Comprimento do pêndulo | $t_1$ (s) | $t_2$ (s) | $t_3$ (s) |

| $l = 90\,\text{cm}$ (uma arruela) | | | |

| $l = 90\,\text{cm}$ (três arruelas) | | | |

*(Tempo referente a 10 oscilações completas em cada medida.)*

O objetivo deste procedimento é investigar se o período $T$ depende da massa $m$ do pêndulo, mantendo fixo o comprimento $l = 90\,\text{cm}$.
