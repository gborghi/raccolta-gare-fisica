# OII 2011 Nazionale Teorica — Problema 4

Quesito n. 4.
Combinando i risultati dei due punti precedenti
$$v^2_{n+1} - v'^2_n = \frac{4MgR\,\text{sen}\,\alpha}{M + Nm} \quad\text{e}\quad v'_n = \left(1 - \frac{1}{2}\frac{m}{M + Nm}\right)v_n$$
si ricava
$$v^2_{n+1} = \left(1 - \frac{1}{2}\frac{m}{M + Nm}\right)^2 v^2_n + \frac{4MgR\,\text{sen}\,\alpha}{M + Nm}$$
e con la relazione data nel testo $M/m = 8$, si ottiene
$$v^2_{n+1} = \left(1 - \frac{1}{2}\frac{1}{8 + N}\right)^2 v^2_n + \frac{32}{8 + N}gR\,\text{sen}\,\alpha$$
Raggiungere una velocità limite significa $v_{n+1} = v_n = v_\ell$ e dalla espressione precedente si ricava
$$v^2_\ell = \left(1 - \frac{1}{2}\frac{1}{8 + N}\right)^2 v^2_\ell + \frac{32}{8 + N}gR\,\text{sen}\,\alpha$$
da cui
$$v_\ell = (64 + 8N)\sqrt{\frac{2gR\,\text{sen}\,\alpha}{(31 + 4N)(8 + N)}}$$
Sostituendo questo risultato nella relazione tra $v'_n$ e $v_n$ ricavata nel quesito precedente, si ottiene
$$v'_\ell = (60 + 8N)\sqrt{\frac{2gR\,\text{sen}\,\alpha}{(31 + 4N)(8 + N)}}$$
