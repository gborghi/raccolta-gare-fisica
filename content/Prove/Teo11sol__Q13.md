# OII 2011 Nazionale Teorica — Problema 13

Quesito n. 2.
Per quanto sopra, dette $Q_1$, $Q_{sp}$, $Q_2$ le cariche superficiali e spaziale, deve essere
$$Q_1 + Q_{sp} + Q_2 = 0 \quad\Rightarrow\quad Q_{sp} = -Q_2 - Q_1 = 4\pi\epsilon_0\,((2R)^2 - R^2)\,E_0 = 12\pi\epsilon_0 R^2\,E_0$$
La carica spaziale (o volumetrica) occupa il volume $V = (4/3)\pi((2R)^3 - R^3) = (28/3)\pi R^3$ per cui la densità
media è
$$\langle\rho\rangle = \frac{Q_{sp}}{V} = \frac{12\pi\epsilon_0 R^2\,E_0}{(28/3)\pi R^3} = \frac{9\,\epsilon_0\,E_0}{7R}$$
