## Problema 4







Find all $k > 0$ for which a strictly decreasing function $g : (0, +\infty) \to (0, +\infty)$ exists such that

$$g(x) \geq k\, g\!\bigl(x + g(x)\bigr)$$

for all positive $x$.

## Problema 6







We call a positive integer $q$ a *convenient denominator* for a real number $\alpha$ if

$$\left|\alpha - \frac{p}{q}\right| < \frac{1}{q^2}$$

for some integer $p$. Prove that if two irrational numbers $\alpha$ and $\beta$ have the same set of convenient denominators, then $\alpha + \beta$ or $\alpha - \beta$ is an integer.
