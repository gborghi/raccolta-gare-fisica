# INJSO 2009 — Problema 58

- (a) Tree → beetles → copepods → fishes → birds
- (b) Tree → copepods → beetles → fishes → birds
- (c) Algae → birds → copepods → beetles → fishes
- (d) Algae → copepods → beetles → fishes → birds
